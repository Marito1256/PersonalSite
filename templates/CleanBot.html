Discord Moderation and Meme Bot

JavaScript JSON Discord library

Project Overview

Discord bot I built in 2019 to moderate swear words in sfw channels. I had to update the bot's programming in 2025 due to deprecated discord functions and an outdated command structure.





Technical Design

The bot is programmed entirely in javascript and runs on NodeJS. Deployed using Docker to my personal server. Secrets are kept in a .env file which is included in the .gitignore to prevent leaked secrets. Dictionary of swear words was retrieved from http://www.bannedwordlist.com/

Use Cases

  • Moderates user messages to detect inappropriate language on sfw channels
  • Includes some fun commands to promote friendly banter without profane or bigoted language
  • Helps to keep track of aggressive users

Technical Implementation

  • Microservices Architecture: Modular design for easy maintenance and scalability.
  • Azure Key Vault: Secure storage and management of secrets, keys, and certificates.
  • Microsoft SQL Server: Reliable database management system for data storage.
  • ASP.NET Core: Utilized alongside SQL Server and Azure Key Vault for robust application development.
